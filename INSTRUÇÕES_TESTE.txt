COMO TESTAR - OPÇÃO 1 (RECOMENDADA):
════════════════════════════════════

1. Abrir em browser desktop: http://localhost/oagb/test-scroll-effect.html

2. Observar a caixa no canto inferior direito com:
   - Scroll Y (valor em pixels)
   - Status do Navbar Scrolled (verde quando ativo)
   - Status do Topbar Scrolled (verde quando ativo)

3. Fazer scroll para baixo:
   ✅ Quando scroll > 45px, as classes devem ser adicionadas (verde)
   ✅ Navbar muda para fundo branco com blur
   ✅ Logo diminui de 70% para 50% de largura
   ✅ Texto do menu fica dourado (#B1A276)
   ✅ Topbar fica branco com texto dourado

4. Fazer scroll para cima:
   ✅ Quando scroll < 45px, as classes devem ser removidas (vermelho)
   ✅ Todos os efeitos devem reverter

   - Verificar que .topbar-scrolled tem background branco

SE NÃO FUNCIONAR:
═════════════════

1. Verificar largura do browser:
   - F12 → Verificar largura (deve ser > 992px)
   - Efeito só funciona em desktop

2. Limpar cache:
   - Windows: Ctrl+Shift+R
   - Mac: Cmd+Shift+R
   - Garantir que CSS/JS não estão em cache

3. Verificar ficheiros:
   - css/header-styles.css (603 linhas)
   - js/header-functions.js (217 linhas)
   - Ambos devem estar nos includes do index.php e apresentacao-historia.php

4. Console JavaScript (F12 → Console):
   - Executar: document.querySelector('.navbar-dark')
   - Deve retornar o elemento navbar
   - Executar: document.querySelector('.bg-dark.px-5.d-none.d-lg-block')
   - Deve retornar a topbar

5. Testar scroll manualmente:
   - Abrir DevTools → Console
   - Executar: window.scrollY
   - Fazer scroll e executar novamente para verificar se o valor muda
   - Executar: document.querySelector('.navbar-dark').classList
   - Deve conter 'navbar-scrolled' quando scroll > 45px
