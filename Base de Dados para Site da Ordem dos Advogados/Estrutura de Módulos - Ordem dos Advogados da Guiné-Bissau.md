### 6.3 Avaliação de Estágios

- Parecer Geral
- Recomendações

## Relações Principais entre Módulos

- **Bastonário** → Órgãos Diretivos (1:N)
- **Comissões** → Membros/Advogados (N:N)
- **Formações** → Inscrições (1:N)
- **Formações** → Estágios (1:N)
- **Estágios** → Avaliações (1:N)
- **Estágios** → Relatórios (1:N)
- **Advogados** → Comissões (N:N)
- **Advogados** → Especialidades (N:N)
- **Eventos** → Inscrições (1:N)
- **Utilizadores** → Permissões (N:N)
