    WEB SPEECH API (readAloud)
    • Objeto: SpeechSynthesisUtterance
    • Idioma: pt-PT (Portuguese Portugal)
    • Callbacks: onstart, onend, onerror
    • Storage: Estado em variável global isSpeaking
    • Targeting: .container div (conteúdo apenas)
    • Elements: p, h3, h4, h5, li (visíveis)

    HEADER INCLUDE SYSTEM
    • Arquivo: includes/header-variant-b.php
    • Parâmetros: $page_title, $breadcrumbs, $background_image
    • Formato Breadcrumbs: array com ['label', 'url']
    • Responsive: Bootstrap (breakpoint 992px)
    • Componentes: Navbar, Breadcrumbs, Background, Quick Actions

    GOOGLE TRANSLATE API
    • CDN: translate.google.com/translate_a/element.js
    • Método: Global google.translate.TranslateElement()
    • Error Handling: Script load verification
    • Positioning: top: 60px, right: 20px
    • Languages: PT (default), EN, FR, ES
